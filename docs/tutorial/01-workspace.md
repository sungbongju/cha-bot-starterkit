# Chapter 1 — 작업 공간 만들기 (`C:\dev`)

> 첫 번째 챕터입니다. 5분 정도 걸려요. 화면만 두 번 클릭하면 됩니다.

---

## 왜 `C:\dev` 폴더부터 만들까요?

코드 프로젝트는 **짧고 깔끔한 폴더 경로**에 두는 게 편합니다.

| 좋은 예 | 나쁜 예 |
|---|---|
| `C:\dev\my-bot\` ✅ | `C:\Users\홍길동\Documents\내 문서\프로젝트 모음\봇만들기\` ❌ |

**이유**:
- 한글이 들어있으면 명령어가 가끔 깨집니다
- 띄어쓰기가 있으면 명령어 입력할 때 따옴표 일일이 넣어야 합니다
- 경로가 길면 윈도우가 가끔 "경로가 너무 길어요" 에러를 냅니다
- `C:\dev` 같이 **루트(C:) 바로 아래 짧은 영어 이름**이 가장 안전

그래서 모든 코드 프로젝트는 `C:\dev\` 폴더 안에 모아두는 게 관례입니다.

---

## 만드는 방법 — 3가지 중 택1

### 방법 A. 파일 탐색기로 (가장 쉬움) ⭐ 추천

1. 키보드에서 **윈도우 키 + E** 누르기 → 파일 탐색기 열림
2. 좌측 패널에서 **내 PC** 클릭 → **로컬 디스크 (C:)** 더블클릭
3. 빈 공간 우클릭 → **새로 만들기 → 폴더**
4. 폴더 이름을 `dev` 라고 입력 → 엔터

→ 완료! 이제 `C:\dev` 폴더가 생겼습니다.

### 방법 B. PowerShell 명령어 (한 줄)

1. 시작 메뉴 → **PowerShell** 검색 → 실행
2. 입력:
   ```powershell
   mkdir C:\dev
   ```
3. 엔터

> 이미 폴더가 있으면 "이미 존재합니다" 에러가 뜨는데 무시해도 됩니다. 폴더가 있으면 OK.

### 방법 C. cmd 명령어

1. 시작 메뉴 → **cmd** 검색 → 실행
2. 입력:
   ```cmd
   md C:\dev
   ```

---

## 확인하기

파일 탐색기 주소창에 `C:\dev` 입력 후 엔터 → 빈 폴더가 열리면 성공.

또는 PowerShell에서:
```powershell
dir C:\
```
출력된 목록에 `dev` 폴더가 있으면 됩니다.

## 🛟 자주 묻는 질문

**Q. 회사/학교 PC라서 C 드라이브에 폴더를 못 만들어요.**
> 그러면 `D:\dev` 또는 `C:\Users\본인이름\dev` 도 OK. 짧고 한글/공백 없으면 됩니다. 아래 자습서에 나오는 `C:\dev` 부분을 본인 경로로 바꾸시면 돼요.

**Q. 이미 다른 곳에 코드 폴더가 있어요.**
> 그대로 쓰셔도 됩니다. 단, 이 자습서의 모든 `C:\dev\` 경로를 본인 경로로 머릿속에서 바꿔주세요.

**Q. Mac/Linux 사용자입니다.**
> `~/dev` 폴더를 만드세요. (`mkdir ~/dev`) 이 자습서의 `C:\dev`는 `~/dev`로 읽어주시면 됩니다.
