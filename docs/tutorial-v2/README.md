## 이 자습서가 약속하는 것

1. **CMD(Command Prompt) 우선** — 화려한 GUI 도구 대신, 어디서나 통하는 명령줄 한 줄로.
2. **풀네임 병기** — 영어 약어는 처음 등장 시 풀네임과 한국어 뜻을 함께 표기합니다.
3. **비유 동반** — 추상 개념마다 일상 비유(쇼핑·도서관·앨범 등)를 동반합니다.
4. **함정 표시** — 봉주가 직접 겪은 막힘 지점을 [참고] 박스로 미리 안내합니다.
5. **검증 표** — 각 장 끝에 "여기까지 잘 됐는지" 확인 표를 둡니다.
