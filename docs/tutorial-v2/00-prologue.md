# 프롤로그 — 이 자습서에 대하여

---

## 왜 만들었나

박대근 교수님께서 학생들에게 강조하시는 말씀이 있습니다.

> "쉽게 가르치자. 학생이 끝까지 따라올 수 있게 하자."

이 자습서는 그 말씀에 대한 봉주(작성자)의 답입니다. 코드를 처음 만져보는 학생이 **막히지 않고 끝까지 완주**할 수 있도록, 그래서 마지막에 친구에게 "야, 내가 만든 AI 봇이야"라며 URL(Uniform Resource Locator, 인터넷 주소)을 보낼 수 있도록 만들었습니다.

봉주 본인도 처음에는 막히는 곳이 많았습니다. `npm install`이 멈춘 줄 알고 Ctrl+C를 눌렀고, 파일 탐색기로 아바타 파일을 옮기다가 엉뚱한 폴더에 떨어뜨렸고, GitHub(깃허브) push가 거부되는 이유를 한참 헤맸습니다. **그 모든 함정을 박스로 모아두었습니다.** 봉주가 한 실수를 여러분이 반복하지 않도록.

## 누구를 위한 것인가

이 자습서가 도와드리는 분들:

- **차의과학대학교 학생** — 박교수님의 수업/세미나에서 이 스타터킷을 처음 만져보는 분
- **코드 경험이 거의 없는 입문자** — "터미널이 뭔지 모르겠다"는 분도 환영
- **수업 자료로 활용하려는 교수님** — 학생들에게 그대로 나눠줄 수 있는 단계별 가이드가 필요한 분
- **취미 개발자** — 주말에 자기만의 캐릭터 봇을 띄워보고 싶은 분

다음 분들은 이 자습서를 일부만 보셔도 됩니다:

- **이미 Git·Node를 다뤄본 분** — 1장(준비물)은 스킵, 4장(`git clone`)부터 보세요.
- **Vercel(버셀) 배포 경험이 있는 분** — 1차 자습서 전체를 30분 안에 훑고 2차로 가시면 됩니다.

## 어떻게 쓰여 있나

### CMD(Command Prompt, 명령 프롬프트) 우선

화려한 GUI(Graphical User Interface, 그래픽 사용자 인터페이스) 도구 대신, **검은 창에 명령어 한 줄을 치는 방식**을 우선합니다. 이유는 세 가지:

1. **어디서나 똑같이 동작합니다.** GUI 도구는 버전에 따라 화면이 바뀝니다.
2. **복사·붙여넣기로 정확히 따라하기 쉽습니다.** "버튼 위치가 어디였더라" 헤매지 않습니다.
3. **실무에서도 명령어가 표준입니다.** 어차피 한 번은 배워야 하는 것, 처음부터 배우자는 취지.

> 박교수님께서도 동의해주신 방침입니다. 처음엔 어색하지만, 3장만 따라하시면 익숙해집니다.

### 비유로 추상 개념 설명

`git add`, `git commit`, `git push` 같은 개념은 처음 들으면 추상적입니다. 그래서 모든 새 개념마다 **일상 비유**를 동반합니다.

- Git의 3단계 → 온라인 쇼핑(장바구니 → 결제 → 배송)
- `git clone` → 도서관에서 책을 복사해 오는 일
- Node.js → "내 컴퓨터에서 자바스크립트를 돌리는 발전기"
- npm(Node Package Manager) → "프로젝트가 필요로 하는 부품을 자동으로 받아주는 택배"

### [참고] 박스

본문에 다 넣기에는 깊거나 부수적이지만 알면 좋은 정보는 **[참고] 박스**로 분리했습니다. 시간이 없으면 건너뛰셔도 본 진도에는 영향이 없습니다.

### 검증 표

각 장 끝에 "여기까지 잘 됐는지" 확인할 수 있는 표를 둡니다. 표의 모든 항목이 ✅이면 다음 장으로 넘어가도 안전합니다.

## 어떤 약속을 지키나

### 약어는 처음 등장 시 풀네임 병기

- 첫 등장: `Git(분산 버전 관리 시스템)`, `npm(Node Package Manager)`, `CLI(Command-Line Interface, 명령줄 인터페이스)`
- 이후 등장: 그냥 `Git`, `npm`, `CLI`

영어 줄임말만 던지지 않습니다. "무슨 뜻인지는 모르겠지만 일단 친다"는 학습을 피하고 싶기 때문입니다.

### 예시 코드는 복사·붙여넣기 가능한 형태로

명령어는 한 줄씩 별도 코드 블록으로 분리합니다. **그대로 복사해서 CMD에 붙여넣으면 동작합니다.**

### 기대 출력 동반

명령어를 친 뒤 어떤 화면이 떠야 정상인지 함께 표기합니다. 화면이 다르면 "뭐가 잘못됐다"는 신호로 곧장 알 수 있습니다.

### 봉주가 겪은 함정은 박스로 강조

```
> 봉주가 직접 막혔던 부분입니다 ...
```

이런 박스를 만나면 "여기는 주의해서 진행"이라는 뜻입니다.
